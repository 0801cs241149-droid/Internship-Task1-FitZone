# 🏋️ FitZone - Online Gym Landing Page

FitZone is a modern, responsive one-page Online Gym website developed using **HTML5** and **CSS3**. It is designed as a clean and attractive landing page for a virtual fitness platform while showcasing modern web design principles.

## ✨ Features

- 🏠 Responsive Navigation Bar
- 💪 Hero Section with Call-to-Action
- ⭐ Feature Cards
- 🏋️ Weekly Workout Schedule
- 🥗 Daily Diet Plan
- 💳 Membership Plans
- 📅 Attendance Tracker
- ⚖️ Weight Progress Tracking
- 🎯 Weight Loss & Weight Gain Goals
- 📞 Contact Section
- 📱 Fully Responsive Design
- 🎨 Modern UI with Hover Effects
- 🔥 Google Fonts & Font Awesome Icons

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome
